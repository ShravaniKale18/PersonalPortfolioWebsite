# Shravani Kale | Personal Portfolio

A modern, highly responsive, and beautifully designed personal portfolio website built to showcase my web development skills, projects, and professional background.

![Portfolio Preview](images/hero_profile.png)

## 🌐 Live Demo
You can view the live website here: [**Live Portfolio URL**](https://ShravaniKale18.github.io/PersonalPortfolioWebsite/)

## ✨ Key Features
- **Modern Dark Theme Aesthetic**: Built with a sleek dark mode featuring glowing gradients, glassmorphism, and neon UI accents.
- **Fully Responsive**: Adapts seamlessly to all devices (Desktops, Tablets, and Mobile phones).
- **Mobile Hamburger Menu**: Smooth, collapsible dropdown navigation for smaller screens.
- **Scroll Animations (Scroll Spy)**: Elements fade up elegantly as you scroll down the page, and the active section highlights automatically in the navigation bar.
- **Functional Contact Form**: Integrated with Formspree for async form submission—meaning messages are sent directly to email without redirecting the user away from the page!
- **Downloadable CV**: Quick access to download my latest resume.

## 🛠️ Technologies Used
- **HTML5**: For the semantic structure of the website.
- **CSS3**: For advanced styling, variables, flexbox/grid layouts, animations, and media queries.
- **JavaScript (Vanilla)**: For DOM manipulation, scroll intersection observers, mobile menu toggling, and `fetch` API form handling.
- **FontAwesome**: For UI icons.
- **Google Fonts**: Uses the modern "Poppins" font family.

## 🚀 How to Run Locally

Since this is a static website, no heavy backend servers or package managers are required to run it!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShravaniKale18/PersonalPortfolioWebsite.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd PersonalPortfolioWebsite
   ```
3. **Open `index.html`** in any web browser of your choice.
